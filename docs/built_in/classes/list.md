# List object documentation

```
List():
    A list which holds multiple objects inside it in an order.

    null push(item):
        Pushes item to the top of the list.
    
    null insert(index, item):
        Inserts the given item at a specific index in the list.
    
    any pop():
        Removes the last item in the list and returns it.
    
    any remove(index):
        Removes an item in a specific index, and returns it.
```
