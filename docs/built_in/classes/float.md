# Float object documentation

```
Float():
    A float, which is one number with decimal points for precision.

    int floor():
        Returns the integer form of the float, while rounding the decimals down (removing them).

        Example: 7.8 becomes 7.
    
    int ceil():
        Returns the integer form of the float, while rounding the decimals up.

        Example: 4.3 becomes 5.
```
